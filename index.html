Contexto

¿Quiénes somos?

Andromeda Space Travel es la estación para aeronaves espaciales más grande de la Tierra tomando en cuenta el número de pasajeros transportados, el número de empleados y número de aeronaves espaciales.
Es también una de las estaciones más antiguas ofreciendo conexiones a todos los planetas de la Vía Láctea.

Problemática

  • No tener una plataforma para la venta de boletos
  • No tener una plataforma para el registro de usuarios
  • No tener una plataforma para el registro de vuelos

Solución

Features

  • Log in

  • Selección por País

Retos

Durante la realización del proyecto surgieron algunos contratiempos que fueron resueltos gracias al trabajo en equipo, investigación, documentación y ...reiniciar los servicios.
Algunos de estos contratiempos fueron:

  • Problemas con las Rutas
  • No tomaba el Token
  • Problemas para instalar Cors
  • Problemas de conexión con la API

Aprendizaje obtenido

Solucionar errores en el código fue difícil. Esto requirió de ingenio y pruebas continuas.

Experiencias

Hubo partes del proyecto que creíamos tardarían más en completarse y salían rápido, sin embargo, cosas más simples podían generar mayor conflicto.
